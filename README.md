\# DondeEntreno



DondeEntreno es una plataforma web pensada para encontrar deportes, clubes, profesores y actividades deportivas dentro de una ciudad.



\## Objetivo del proyecto



El objetivo principal es crear una aplicación full stack que permita:



\- Buscar actividades deportivas.

\- Ver clubes disponibles.

\- Encontrar profesores.

\- Filtrar por deporte, ubicación, horarios y modalidad.

\- Contactar fácilmente con clubes o profesores.
