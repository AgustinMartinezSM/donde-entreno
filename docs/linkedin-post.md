# Post LinkedIn - DondeEntreno

Después de varias etapas de desarrollo, deploy y pruebas, ya tengo online el MVP de **DondeEntreno**.

DondeEntreno es una plataforma web pensada para ayudar a las personas a encontrar deportes, clubes, profesores y actividades deportivas dentro de una ciudad.

La idea surgió de una situación bastante común: querer empezar a entrenar, buscar un deporte específico o llegar a una ciudad nueva y no saber bien dónde encontrar opciones reales para practicar una actividad.

Muchas veces esa información está dispersa entre redes sociales, recomendaciones, Google Maps o publicaciones sueltas. Por eso, el objetivo de DondeEntreno es centralizar todo en un solo lugar y hacer más simple el proceso de buscar, comparar y encontrar dónde entrenar.

En este MVP trabajé en:

* Base de datos relacional con PostgreSQL.
* Backend con Java 21 y Spring Boot.
* API REST para actividades, filtros y detalles.
* Frontend con Next.js, TypeScript y Tailwind CSS.
* Deploy full stack usando Supabase, Render y Vercel.
* Documentación técnica, testing manual y feedback inicial de usuarios.

Stack utilizado:

```text
Frontend: Next.js, React, TypeScript, Tailwind CSS, Vercel
Backend: Java 21, Spring Boot, Spring Data JPA, Maven, Docker, Render
Base de datos: PostgreSQL, Supabase
Repositorio: GitHub
```

Proyecto online:

Todavía es un MVP, pero ya está funcionando online y fue compartido con algunas personas para recibir feedback real.
